# 介绍

随机推荐 NPM 库

# 技术栈

- 前端：TypeScript + React + Antd
- 后端：Koa
- 数据库：MongoDB

## 数据库设计

- 数据库名：bookmark

### NPM 收藏集合 —— NPM 文档

| NAME        | REMAKRS  |
| ----------- | -------- |
| name        | 名称     |
| description | 描述     |
| create_time | 添加时间 |
| update_time | 更新时间 |
| link        | 链接     |
| category    | 分类     |
| tag         | 标签     |
