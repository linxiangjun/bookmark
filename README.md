# 介绍

根据自定义配置从NPM爬取数据并随机展示还没有阅读过的高质量NPM package

# 背景

如果每天阅读10个高质量的NPM package，那么一个月你将累计300个package的阅读量，这样会大大帮助你提升遇见问题的解决能力（是造轮子还是用现成的，什么样的package合适）。
